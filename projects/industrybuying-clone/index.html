Overview

A full-stack e-commerce clone built during an immersive bootcamp, covering the complete CRUD product-catalog lifecycle.

The Challenge

Recreate a B2B e-commerce experience end-to-end, including a REST API and database, on a tight team deadline as a group project.

The Approach

Designed the MongoDB schema for products and orders, built Express REST endpoints for full CRUD, and implemented the storefront UI in React with Chakra UI for consistent, accessible components.

The Result

Delivered a working full-stack e-commerce app with complete create/read/update/delete flows, deployed and demoed within the bootcamp sprint deadline.
